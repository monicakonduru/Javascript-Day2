# Javascript-Day2

Day 2 of coding consistently and focusing on strengthening my fundamentals by building small projects from scratch without relying on AI-generated code.

## Project

A simple Todo List application built using React and Tailwind CSS.

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Track completed tasks count

## Concepts Practiced

- React functional components
- `useState` hook
- State management
- Event handling
- Array methods like `map()` and `filter()`
- Conditional rendering
- Tailwind CSS basics

## Purpose

The goal of this project is to improve core JavaScript and React fundamentals through consistent practice and small daily builds.

## Future Improvements

- Edit tasks
- Store todos using localStorage
- Add task filters
- Improve responsiveness and accessibility

## Tech Stack

- React
- JavaScript
- Tailwind CSS
